BeeStation Product launch Website

Website Design

BeeStation 是一款為科技小白打造的初階版 NAS，結合落地備份與雲端功能。不僅省去了繁多的硬碟設備，還能有效防止資料遺失和網路病毒的風險。
在網頁設計策略上捨棄了傳統的科技感風格，而是採用了更貼近家居生活的設計。這樣的風格讓消費者感受到，BeeStation 不是一個複雜難懂的科技產品，提升網站參與度。
為了突出 BeeStation 的核心價值——「易上手」與「便利性」，我們也將軟體內的 UI 結合網站動畫，無論是哪一個語系都可以快速且簡單的了解BeeStation。

Visit BeeStation website >

BeeStation by Synology is a hard disk with a network connection. It combines local and cloud backup features, eliminating the need for multiple hard drives while significantly reducing the risks of data loss and network security threats.
The website design strategy uses a style inspired by furniture and household items.By moving away from traditional tech-focused aesthetics, the design makes the product feel approachable and increases website engagement.
To highlight the core features of BeeStation are easy to use and convenient. We’ve integrated the software UI with website animations. This ensures that users, regardless of their language, can quickly and easily understand BeeStation.

BeeStationは、Synologyが開発したネットワーク接続対応のハードディスクです。ローカルとクラウドバックアップ機能をひとつに統合し、複数のハードディスクを必要とせずに、データ損失やネットワークセキュリティ上のリスクを大幅に軽減します。
ウェブサイトデザインは、従来のテクノロジー感のあるデザインを避け、製品に親しみやすさを感じてもらい、より多くの人がサイトに興味を持ち、滞在時間が増えるように工夫されています。
BeeStationの核心価値である「使いやすさ」と「利便性」を際立たせるため、ソフトウェアのUIとウェブサイトのアニメーションを組み合わせました。どの言語のユーザーでも、BeeStationを速やかに理解できるように作り上げています。
